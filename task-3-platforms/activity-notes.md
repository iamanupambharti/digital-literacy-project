# Activity Notes

## Platforms Used
*   **HackerRank:** This was used to practice basic programming logic and complete the 'Solve Me First' challenge.
*   **Google Forms:** This was used to develop a digital literacy assessment tool for peer learning.
*   **Google Sheets:** This was used to organize and analyze data obtained from the awareness quiz.

## Reflection
For this task, I used platforms that assist in developing technical skills and collaborating in the cloud. I used HackerRank to complete a beginner-level programming challenge, which assisted me in getting accustomed to the environment used for technical interviews and competitions.

Moreover, I developed a five-question 'Digital Literacy Awareness Quiz' using Google Forms to test my batchmates on internet safety and professional communication skills. These tools will prove to be extremely helpful for my academic path at VIT Bhopal.

Learning HackerRank will allow me to monitor my progress with data structures and algorithms over the course of four years. Moreover, learning Google Workspace, particularly the integration between Google Forms and Sheets, is necessary for collaborating on group assignments.

These exercises have shown me that digital literacy is not only about learning an individual skill, but it is also about using the correct tools to disseminate information, particularly with other people within a cloud-based system.
